dfs 구현 방법 
<br>
스택을 만든다.
```
Stack<integer> stac = new Stack<>();
```

꺼내서 하나씩 조건에 맞는 값을 스택에 넣는다.
```
while(!stac.isEmpty){
    int n = stac.pop();
    //로직
}
```

여기서는 탈출 조건을 원하는 길이의 소수 값을 찾을 때 까지로 진행했다.
```
if(n.length() == size){
```
